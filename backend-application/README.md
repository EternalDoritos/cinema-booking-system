# Backend code for Cinema Booking System

## Dependencies and frameworks used
- Node
- Express
- DotEnv
- Mongoose
- bcrypt

## Instructions to start up backend
- Ensure that the env file is present (env file is not uploaded to github)
- From terminal, navigate to this folder and initiate node with npm start
- Once connected, your terminal should display node being connected on port 5000 and mongodb is connected
